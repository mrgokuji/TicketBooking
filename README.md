# TicketBooking
Ticket Booking system for movie screens.

Clone the repo
git clone https://github.com/mrgokuji/TicketBooking.git

# Prerequisite
- Java 11
- npm 6.14.15
- node v14.18.1


# Start Backend Server
- cd ticket
- ./mvnw spring-boot:run -X

# Start react Frontend
- npm install
- npm start
